# Api Rest movies
api rest que permite almacenar las mejores peliculas del mercado.


## Instalación

```shell
    $ git clone https://github.com/jorcalle11/api-rest-ES6.git
    cd api-rest-ES6.git
    npm install
```

## Ejecución

```shell
    $ npm start
```    
## Pruebas

```shell
    $ make test
```
